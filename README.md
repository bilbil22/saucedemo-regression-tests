# SauceDemo Regression Tests

This project demonstrates a regression test suite for SauceDemo using Python and Playwright.

Due to time constraints, the focus was on:
- covering key user flows (login, cart, checkout)
- establishing a scalable structure (Page Object Model)
- demonstrating automated regression capability

The suite includes both core scenarios and extended coverage.

During development, I focused on understanding Playwright and structuring maintainable test automation rather than only maximizing test count.

Given more time, I would:
- refine test data management
- improve test stability and selectors
- integrate deeper CI/CD pipelines
